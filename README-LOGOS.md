# Instruções para adicionar as logos

Por favor, adicione os seguintes arquivos de imagem na raiz do projeto:

1. **logo-ig.png** - Logo do IG (Instituto de Geociências)
2. **logo-unicamp.png** - Logo da Unicamp

Esses arquivos devem estar na mesma pasta que o index.html.

Se preferir usar SVG para melhor qualidade, nomeie como:
- logo-ig.svg
- logo-unicamp.svg

E ajuste as referências no index.html de .png para .svg
